Cebuano Natural Language Toolkit

Cebuano Natural Language Toolkit is built around the Python programming language since Natural Language Processing with Python provides a practical introduction to programming for language processing works. The NLTK book (https://www.nltk.org/book/) guides the reader through the fundamentals of writing Python programs, working with corpora, categorizing text, analyzing linguistic structure, and more.

In this page users are able to test sample sentences written in the Cebuano texts that will be then passed to the CNLTK model and will return the Preprocessed and Tagged sample sentence. After going through the website, filling out the feedback form would be much appreciated.

Part of Speech Tagger

The Part Of Speech (POS) Tagger tool assigns parts of speech to each word in a sentence, such as noun, pronoun, verb, adverb, and adjective. A POS tagger's primary function is to provide grammatical information about a text, which is required for a variety of natural language processing (NLP) tasks. The development of this POS Tagger tool, particularly because it uses Cebuano written data, can help to advance NLP applications in that language. This is especially true for languages with limited resources and support for NLP applications. The creation of a POS Tagger tool will aid in the development of language-specific NLP models for use in a variety of applications such as sentiment analysis, machine translation, and text classification.

Named Entity Recognition

The Named Entity Recognition tagger is a tool that uses NER techniques to identify named entities in a given text, such as people, organizations, locations, and so on. A NER tagger's primary function is to extract structured information from unstructured text data. The development of this NER tagger, particularly because it uses Cebuano written data, can help to advance NLP applications in that language. This is especially true for languages with limited resources and support for NLP applications. The creation of a NER tagger tool will aid in the creation of language-specific NLP models for use in a variety of applications such as information retrieval, question answering, and event extraction.
