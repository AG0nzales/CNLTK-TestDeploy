Cebuano Natural Language Toolkit

CNLTK is a platform alike NLTK which serves as a guide and demo, it is also a platform for building Python programs to work with Cebuano language data. It provides easy-to-use interfaces, current available corpora sourced from Newspaper articles and Medical texts in the Cebuano language, along with a suite of text processing libraries. Thanks to hands-on guides introducing programming fundamentals alongside topics in computational linguistics, plus comprehensive API documentation, CNLTK is suitable for linguists, engineers, students, educators, researchers, and industry users alike which plays around with Cebuano language datasets.

CNLTK is inspired to NLTK which has been called “a wonderful tool for teaching, and working in, computational linguistics,” and “an amazing library to play with natural language.”
